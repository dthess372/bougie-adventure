# Brand Assets

Source and print files for the Bougie Adventure logo. These are NOT served on the website; web-ready versions live in `public/` (`logo.png`, `apple-icon.png`).

- `logo-color.ai` / `logo-bw.ai`: Illustrator source files
- `logo-*-print-cmyk.jpg`: CMYK print versions (will look wrong in browsers; for printers only)
- `logo-*-transparent.png` / `logo-*-white-bg.png`: 2100px RGB masters

To regenerate the web logo, trim the transparent padding from `logo-color-transparent.png` and resize to 1024x1024.
